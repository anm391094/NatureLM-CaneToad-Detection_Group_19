NatureLM-audio based cane toad (Rhinella marina) vocalisation detection using bioacoustic recordings from the Australian Acoustic Observatory.
# NatureLM Cane Toad Detection

## Project Overview

This project evaluates NatureLM-audio for detecting
Rhinella marina (Cane Toad) vocalisations using
recordings from the Australian Acoustic Observatory.

## Environment

- Google Colab
- A100 GPU
- Python 3

## Dataset

22,006 labelled audio recordings

## Installation

pip install -r requirements.txt

## Running

Open main_gc_file.ipynb

Follow cells sequentially.

## Outputs

Results are saved to Google Drive outputs folder.
